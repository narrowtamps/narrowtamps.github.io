# d20-icosahedron

A Pen created on CodePen.

Original URL: [https://codepen.io/e-Nicko/pen/LYoOLoz](https://codepen.io/e-Nicko/pen/LYoOLoz).

This project demonstrates an interactive, rotating 20-sided die (icosahedron) using Three.js. Each face of the icosahedron is a triangle, and the edges are highlighted with thick white lines for better visibility. The figure rotates smoothly and can be controlled with mouse interactions thanks to the OrbitControls feature.

Features:
+ Three.js: Utilizes the powerful Three.js library for 3D rendering.
+ Interactive Controls: Use OrbitControls to allow for mouse interaction with the 3D object.
+ Thick Edges: Multiple lines create the effect of thick edges for better visual clarity.
Smooth Animation: Continuously rotating icosahedron for a dynamic display.

Usage:
+ Drag the mouse to rotate the icosahedron.
+ Scroll to zoom in and out.
